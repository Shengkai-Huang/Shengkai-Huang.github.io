# Personal Profile Website

This is a multi-page static personal profile website designed for GitHub Pages.

## Pages

- `index.html` - home page with personal introduction, interests, and page links
- `about.html`
- `skills.html`
- `experience.html`
- `education.html`
- `contact.html`

## Preview locally

Open `index.html` directly in your browser.

## Publish to GitHub Pages

1. Create a public GitHub repository named exactly:

   `yourusername.github.io`

2. Upload everything in this folder:

   - all `.html` files
   - `styles.css`
   - `script.js`
   - `assets/`
   - `README.md`

3. Go to the repository on GitHub.
4. Open `Settings` > `Pages`.
5. Under `Build and deployment`, choose:

   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

6. Your site will be available at:

   `https://yourusername.github.io`
